## thermometer

Термометр-сигнализатор на базе ESP32 с возможностью калибровки и звуковой сигнализацией при выходе температуры за заданные пределы.

**Особенности:**
- Автоматическая фиксация базовой температуры (3 мин стабилизации)
- Настраиваемый верхний порог тревоги
- Фиксированный нижний порог (Base − 0.3°C)
- Режим калибровки с сохранением в EEPROM
- OLED дисплей 128×64 (SSD1306)
- Звуковая сигнализация пассивным зуммером
